# Retail Product Sales Analysis Dashboard

## Project Overview
Developed an interactive Business Intelligence dashboard using Power BI and Excel to analyze retail product sales, costs, profits, and profit margins. The dashboard helps track business performance through interactive visualizations and KPI monitoring.

## Tools Used
- Power BI
- Microsoft Excel

## Features
- Product-wise Sales Analysis
- Category-wise Performance Tracking
- Profit and Profit Margin Analysis
- KPI Monitoring
- Interactive Dashboard Visualizations
- Business Insight Reporting

## Key Insights
- Banana generated the highest sales and profit.
- Fruit category contributed higher overall sales.
- Dashboard helps identify product performance and profitability trends.

## Project Files
- Power BI Dashboard (.pbix)
- Excel Dataset
- Dashboard PDF
- Dashboard Screenshots

## Outcome
The project provides clear business insights and supports data-driven decision-making using visual analytics and reporting.
